# pyrequest

  直接运行run_test.py
  
  若需要加入新的邮箱收件者，则在yanss\sendMail下主函数中receivers加入，逗号隔开

  它包含功能:
  * 测试数据初始化，并对数据的插入做了封装。
  * unittest单元测试框架运行测试
  * HTMLTestRunner生成接口测试报告
  * 以邮件方式发送报告至指定邮箱


Python版本与依赖库：
  * python2.7+ :https://www.python.org/
  * Requests : https://github.com/kennethreitz/requests  
     pip install requests
